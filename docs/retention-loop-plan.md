# Compinche Space — Retention Loop (Daily Engagement)

## Objetivo
Aumentar retención diaria con un loop emocional + productivo:
1. Trigger diario (mensaje + estado)
2. Acción mínima guiada
3. Recompensa inmediata
4. Progreso visible + desbloqueos

## Implementado en la app
- Home tipo dashboard moderno con:
  - saludo personalizado
  - racha diaria
  - % de día completado
  - nivel emocional del usuario
- Rutinas específicas:
  - 😴 Sueño (modo descanso profundo)
  - ✍️ Journaling rotativo
  - 😮‍💨 Anti-estrés 3 min
  - ⚡ Productividad con pomodoro
- Flujo guiado por pasos (transición + barra de progreso + cierre emocional).
- Sistema de niveles:
  - 🌱 Semilla → 🌿 En proceso → 🌸 Floreciendo → ✨ En expansión → 💫 Compinche Pro
- Gamificación avanzada:
  - badges
  - desbloqueos
  - recompensas
- Integración bienestar + productividad:
  - después de foco → respirar
  - después de journaling → siguiente acción
- Modo según energía (baja/alta) que reordena recomendaciones de rutina.

## Próximos pasos (backend + growth)
- Persistir eventos por rutina y energía en Supabase.
- Push notifications segmentadas por nivel y consistencia.
- A/B test de microcopy en CTA principal y cierres.
